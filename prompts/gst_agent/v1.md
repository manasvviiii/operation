Extracts a 15-character alphanumeric GST number pattern from the most recent inbound message using regex /\b\d{2}[A-Za-z]{5}\d{4}[A-Za-z]{1}[A-Za-z\d]{1}Z[A-Za-z\d]{1}\b/. If no match is found, responds asking the vendor to provide their GST number (15-character alphanumeric).
